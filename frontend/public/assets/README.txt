Place your branding images in this folder.

Required file:
- logo.png  (used for header logo and favicon)

Optional files:
- screenshot.png  (if you want to showcase a screenshot in docs or pages)

Instructions (Windows PowerShell):
1) Copy your logo into this folder as logo.png
   Example:
   Copy-Item -LiteralPath "C:\Users\SAPTARSHI MONDAL\Downloads\your-logo.png" -Destination ".\logo.png"

2) If you want to include a screenshot for docs, copy it here as screenshot.png
   Example:
   Copy-Item -LiteralPath "C:\Users\SAPTARSHI MONDAL\OneDrive\Pictures\Screenshots\Screenshot 2025-10-26 060919.png" -Destination ".\screenshot.png"
